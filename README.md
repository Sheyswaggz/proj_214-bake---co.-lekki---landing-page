# Bake & Co. Lekki - Landing Page

A visually appealing and informative landing page for Bake & Co., a premium bakery based in Lekki, Lagos, Nigeria. This page showcases the bakery's brand, highlights its delicious products and services, provides location and contact information, and encourages potential customers to visit or get in touch.

## File Structure

```
.
├── index.html              # Main landing page
├── styles/
│   ├── main.css           # Core styles and design system
│   └── animations.css     # Scroll animations and transitions
├── scripts/
│   └── main.js            # Interactive features and animations
└── README.md              # This file
```

## Local Development

To run this project locally:

### Option 1: Direct File Access
Simply open `index.html` in your web browser.

### Option 2: Local Server
For the best experience, run a local development server:

```bash
# Using Python 3
python3 -m http.server 8080

# Then visit http://localhost:8080 in your browser
```

## Technology Stack

- **HTML5** - Semantic markup structure
- **CSS3** - Modern styling with custom properties
- **Vanilla JavaScript** - Interactive features without frameworks

## Features

- **Responsive Design** - Optimized for all screen sizes from mobile to desktop
- **Scroll Animations** - Smooth reveal effects as users scroll through content
- **Lazy Loading** - Optimized image loading for better performance
- **Warm Bakery Theme** - Inviting color palette and typography that reflects the brand
- **Mobile-First Approach** - Built with mobile users as the primary focus
